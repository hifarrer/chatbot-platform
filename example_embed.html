About This Demo

This page demonstrates how easy it is to add an AI chatbot to your website using owlbee.ai. The chatbot appears as a floating button in the bottom-right corner.

How It Works

  • Create and train your chatbot on owlbee.ai
  • Get your unique embed code
  • Paste the code into your website
  • Your visitors can now chat with your AI assistant!

ðŸ“‹ How to Add This to Your Website

Step 1: Replace the embed code below with your actual chatbot's embed code from owlbee.ai.

Step 2: Update the domain URL to match where your owlbee.ai is hosted.

Step 3: Paste this code before the closing </body> tag on your website.

<!-- ChatBot Embed Code --> <div id="chatbot-YOUR_EMBED_CODE"></div> <script src="https://your-domain.com/static/js/chatbot-embed.js"></script> <script> ChatbotEmbed.init({ embedCode: 'YOUR_EMBED_CODE', apiUrl: 'https://your-domain.com/api/chat/YOUR_EMBED_CODE', containerId: 'chatbot-YOUR_EMBED_CODE', title: 'My AI Assistant', placeholder: 'Ask me anything...' }); </script>
