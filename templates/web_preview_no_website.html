No Website URL Configured

To use the Web Preview feature, you need to add a Homepage URL to your chatbot settings, or add your business website URL to your profile.

How to enable Web Preview:

Option 1 - Chatbot Homepage URL (Recommended):

  1. Click "Edit" on the Chatbot Information section
  2. Enter your Homepage URL in the chatbot settings
  3. Save your changes
  4. Return here to use Web Preview

Option 2 - Profile Website:

  1. Go to your Profile page
  2. Add your business website URL in the Website field
  3. Save your profile
  4. Return here to use Web Preview

Note: The chatbot Homepage URL takes priority over the profile website.
